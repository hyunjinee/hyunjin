# MDX 테스트 페이지

Next.js의 네이티브 MDX 지원을 테스트하는 페이지입니다.

## 기능 테스트

### 1. 텍스트 서식

일반 텍스트, **굵은 글씨**, _기울임꼴_, ~~취소선~~ 등을 테스트합니다.

### 2. 코드 블록

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`)
  return `Welcome to MDX!`
}

greet('World')
```

```typescript
interface User {
  id: number
  name: string
  email: string
}

const user: User = {
  id: 1,
  name: 'John Doe',
  email: 'john@example.com',
}
```

### 3. 인라인 코드

`const variable = 'value'`와 같이 인라인 코드도 사용할 수 있습니다.
